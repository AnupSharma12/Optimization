# Optimization

A simple, no-nonsense website with practical tips to optimize your devices — Windows, macOS, Linux, Android, and iOS.

## What It Does

- Browse optimization tips organized by platform
- Filter by platform (Windows, macOS, Linux, Android, iOS)
- Search tips instantly with a live search bar
- Toggle between light and dark themes
- Fully responsive — works on desktop, tablet, and mobile

## Platforms Covered

| Platform | Categories |
|----------|-----------|
| Windows  | Performance, Storage, Network, Security |
| macOS    | Performance, Storage, Battery |
| Linux    | Kernel & System, Storage, Network, Power |
| Android  | Speed, Battery, Storage |
| iOS      | Performance & Battery, Storage |
| General  | Universal tips for all devices |
| Tools    | Recommended free utilities |

## Tech Stack

- **HTML** — semantic markup, no frameworks
- **CSS** — custom properties for theming, responsive grid layout
- **JavaScript** — vanilla JS for search, filtering, theme toggle, and mobile menu

## How to Use

1. Clone or download this repo
2. Open `index.html` in any browser
3. That's it — no build step, no dependencies
